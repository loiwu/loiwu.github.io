C程序下内存管理的技术

管理固定大小的内存

栈内存：函数中定义的基本类型变量和对象的引用变量都在函数的栈内存中分配
生存域：变量所处{和}之间的区域。当程序执行到}时，会将}对应生存域内的所有变量占用的内存释放给操作系统。

    int main () {
        int array[] = {1,2,3,4,5};
                                ----------------|
        if ( i > 0 ) {                          |
            ...                                 |
            ...                                 |
        }                                       |
        foo(array);                             |
                                                |
        {                                       |
            int k = 9;        ---               |
            array[1] = k;       |->k的生存域     |->array的生存域
            ...               ---               |
        }                                       |
    }                                           |
                                                |
    void foo(int * const number) {              |
        int j = 0;                              |
                           ------               |
        ...                     |               |
        number[2] = j;          |->j的生存域     |
        ...                     |               |
                           ---------------------|
    }
